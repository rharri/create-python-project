A simple script to create a new Python Project.

## Example
```
$ ./pycreate.sh
Name: hello-service
Python Version (ex. 3.12): 3.12
Package Name: hello_service


Next Steps: 
 - Run source .venv/bin/activate
 - Run pip install -r etc/requirements-dev.in
 - Run sphinx-quickstart
 - Packaging
$ cd hello-service
```
